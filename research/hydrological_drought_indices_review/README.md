## Zentrale Erkenntnisse

### Hydrologische Indizes
- **SMI**: Bodenfeuchte-basiert, Percentil oder z-Transformation
- **SSI**: Standardisierte Bodenfeuchte, Gamma-Verteilung
- **SDI**: Abfluss-basiert, kumulierter Zeitreihen-Ansatz
- **SPEI**: Wasserbilanz, log-logistische Verteilung
- **PDSI**: Komplexer Wasserhaushalt, empirisch

### Modelle
- **mHM**: Mesoskalige Hydrologie, SMI-Integration
- **VIC**: Variable Infiltrationskapazität
- **SWAT**: HRU-basiert, Bodenschichten
- **Noah-MP**: Landoberflächenmodell
- **HBV**: Konzeptionelles Modell

### Forschungslücken
1. Optimale Kombination verschiedener Indizes
2. Zeitliche Verzögerungen zwischen Dürretypen
3. Integration von Fernerkundungsdaten
4. Validierung multi-indikativer Ansätze
5. Robustheit unter Klimawandel-Szenarien

## Matrixansatz
