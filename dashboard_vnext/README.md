# Scientific Drought Dashboard (React/TypeScript)

This dashboard implements a multi-index, scientifically structured drought workflow with:

- Spatial drought overview (domain centroid map panel)
- Time-series comparison (SPI, SPEI, SMI, discharge) with dynamic windows
- Hydrological drought propagation (`P -> SM -> recharge -> Q`) and lag correlation
- Extreme drought analysis (threshold exceedances and event timeline)
- Index comparison diagnostics (correlation heatmap + agreement heatmap)
- Validation panel (KGE/NSE/RMSE/MAE/Bias/Peak/Timing + hydrograph)

Implemented indices and data products:

- SPI (1/3/6/12), SPEI-3, SMI
- Hydrological variables for propagation: precipitation, PET, soil moisture, recharge, runoff, discharge
- Validation metrics: KGE primary, NSE, RMSE, MAE, Bias, Peak Error, Timing Error

## Setup

```bash
cd dashboard_vnext
npm install
```

## Build

```bash
npm run build
```

## Dev Server

```bash
npm run dev
```

Default dev URL: `http://localhost:8510`

## Data Source

Generated by:

```bash
cd /data/.openclaw/workspace/open_claw_vibe_coding
/data/.openclaw/workspace/miniforge/bin/python analysis/scripts/build_multi_index_monitor.py --domain all
```

This writes:

- `analysis/results/<domain>/monitor_indices.csv`
- `dashboard_vnext/public/data/<domain>_monitor.json`

## Time Window Modes

- Full record
- Last 30 years
- Selected drought events (2003/2018/2019/2020 presets)
- User-defined custom period
