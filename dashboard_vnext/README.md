# Multi-Index Drought Monitor (React/TypeScript)

Implementation of the Codex prompt `codex_prompt_drought_monitor_LIVE.md` with:

- SPI (1/3/6/12), SPEI-3, SMI
- Validation metrics (KGE primary, NSE, RMSE, MAE, Bias, Peak Error, Timing Error)
- Progressive disclosure UX
- Tabs: Time Series / Comparison / Validation
- 30+ year continuous plotting with zoom/pan
- Comparison views: radar, scatter, parallel coordinates, consistency heatmap
- Tailwind + Apache ECharts

## Setup

```bash
cd dashboard_vnext
npm install
```

## Build

```bash
npm run build
```

## Dev Server

```bash
npm run dev
```

Default dev URL: `http://localhost:8510`

## Data Source

Generated by:

```bash
cd analysis/scripts
python3 build_multi_index_monitor.py --domain all
```

This writes:

- `analysis/results/<domain>/monitor_indices.csv`
- `dashboard_vnext/public/data/<domain>_monitor.json`
